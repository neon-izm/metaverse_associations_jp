# メタバース協会を設立するには？
## 方法１：法務局に登記を行う
最小２名から設立可能です。

https://houmukyoku.moj.go.jp/homu/houjin3.html

## 方法2:メタバース業界団体ガチャ
Twitterアカウントがあれば診断メーカーによって生成可能です。  
https://shindanmaker.com/1114938
